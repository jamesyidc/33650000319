🔄 清除缓存指南

如果您看到"实时数据不完整"或页面一直加载中，请按照以下步骤操作

✅ 方法1：完全清除缓存（最推荐）

步骤1：打开开发者工具

按键盘上的 F12

步骤2：切换到Application标签

在开发者工具顶部找到 Application 标签并点击

步骤3：清除网站数据

在左侧找到 Storage 区域

右键点击域名 → 选择 "Clear site data"

重要：确保勾选所有选项（Cookies、Local Storage、Cache Storage、Service Workers等）

步骤4：关闭并重新打开页面

完全关闭浏览器标签页，然后重新打开

⚡ 方法2：强制刷新（快速但可能不够彻底）

Windows/Linux：

Ctrl + Shift + R

macOS：

Cmd + Shift + R

注意：此方法可能无法清除Service Worker缓存，如果仍有问题，请使用方法1

🕵️ 方法3：隐身模式测试

打开浏览器隐身模式：

Ctrl + Shift + N

在隐身窗口中访问页面，查看是否正常

如果隐身模式正常，说明确实是缓存问题，回到正常模式使用方法1清除缓存

✓ 验证修复是否成功

✅

✅
