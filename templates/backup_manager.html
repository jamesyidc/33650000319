系统概述

这是一个完整的加密货币交易分析系统，集成了多个子系统用于数据采集、分析和交易信号生成。

🔄 实时数据采集

18个数据采集器并行运行

📊 多维度分析

SAR、恐慌指标、支撑压力等

🎯 交易信号

自动生成买卖信号

💾 自动备份

12小时自动备份，保留最近3次

系统架构 

Flask Web Server (端口9002)
    ├── 数据采集器 (18个PM2进程)
    │   ├── coin-price-tracker (币价追踪)
    │   ├── sar-jsonl-collector (SAR指标)
    │   ├── market-sentiment-collector (市场情绪)
    │   └── ... (其他15个)
    ├── 分析系统
    │   ├── SAR Bias统计
    │   ├── 支撑压力分析
    │   └── 恐慌指标分析
    ├── 监控系统
    │   ├── data-health-monitor (数据健康)
    │   └── system-health-monitor (系统健康)
    └── JSONL数据库 (~3.6GB)

核心数据表结构

1. SAR指标数据 (data/sar_jsonl/*.jsonl)

每个币种一个文件，如 BTC.jsonl

{
  "timestamp": "2026-03-15T10:30:00+08:00",
  "coin": "BTC",
  "price": 65432.10,
  "sar": 64500.00,
  "sar_direction": "bull"
}
2. 支撑压力数据 (support_resistance_levels.jsonl) - 697MB

最大文件，记录所有支撑压力位

{
  "timestamp": "2026-03-15T10:30:00+08:00",
  "coin": "BTC",
  "support_levels": [64000, 63500, 63000],
  "resistance_levels": [66000, 66500, 67000],
  "strength": "strong"
}
3. 市场情绪 (market_sentiment_*.jsonl) 
{
  "timestamp": "2026-03-15T10:30:00+08:00",
  "sentiment_score": 6.5,
  "fear_greed_index": 45,
  "market_mood": "neutral"
}
其他数据表: 恐慌清洗指标 (panic_wash_index.jsonl)、 锚点数据 (anchor_daily/*.jsonl, 191MB)、 价格速度 (price_speed_jsonl/*.jsonl, 173MB)、 OKX交易日志 (okx_trading_logs/*.jsonl)、 SAR Bias统计 (sar_bias_stats/*.jsonl)

系统依赖

Python包:
  • Flask 3.0.0 (Web框架)
  • pandas 2.1.4 (数据处理)
  • numpy 1.26.2 (数值计算)
  • ccxt 4.2.0 (交易API)
  • requests 2.31.0 (HTTP)
PM2进程 (18个):
  • flask-app (主应用)
  • coin-price-tracker (币价)
  • sar-jsonl-collector (SAR)
  • market-sentiment (情绪)
  • ...及其他14个进程
数据目录 (~3.6GB):
  • support_resistance_daily (977MB)
  • support_resistance_jsonl (740MB)
  • anchor_daily (191MB)
  • price_speed_jsonl (173MB)
  • anchor_profit_stats (163MB)
环境变量:
  • FLASK_PORT=9002
  • DATA_DIR=/home/user/webapp/data
  • BACKUP_INTERVAL=43200
  • MAX_BACKUPS=3

备份系统

自动备份策略:

• 频率: 12小时自动执行
• 保留: 最近3次备份
• 位置: /tmp/webapp_backup_YYYYMMDD_N.tar.gz
• 压缩比: ~12:1 (3.16GB → 265MB)

备份内容 (3,325个文件):
• Python: 401文件 (5.38MB)
• HTML: 289文件 (14.02MB)
• 配置: 290文件 (3.20MB)
• 数据: 931文件 (2.97GB)
• 日志: 38文件 (1.52MB)
• 其他: 1,359文件 (167.30MB)
