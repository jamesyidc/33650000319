## 🎨 K线颜色标识规则

### 颜色对照表

| Emoji | 颜色 | 条件 | 说明 |
|-------|------|------|------|
| 🟩 | **绿色** | `change_percent > 0` | 收盘价 > 开盘价（上涨））|
| 🟥 | **红色** | `change_percent < 0` | 收盘价 < 开盘价（下跌））|
| ⬜ | **白色** | `change_percent = 0` | 收盘价 = 开盘价（平盘）|

### 计算公式

```python
# K线数据
open_price = 71171.3    # 开盘价
close_price = 71194.1   # 收盘价

# 涨跌幅计算
change_percent = (close_price - open_price) / open_price
# = (71194.1 - 71171.3) / 71171.3
# = 0.00032 = 0.032%

# K线颜色判断
if change_percent > 0:
    color = "🟥 (红色)"  # 上涨K线
elif change_percent < 0:
    color = "🟩 (绿色)"  # 下跌K线
else:
    color = "⬜ (平盘)"  # 平盘K线（极少见）
```

### 修改文件
- **consolidation_monitor.py** - `send_telegram_alert()` 函数

### 关键代码

```python
def send_telegram_alert(symbol, config_name, consecutive_count, records):
    """发送 Telegram 横盘告警"""
    # ... 前面代码 ...
    
    message = f"""🔔 <b>横盘变盘告警</b>

📊 <b>{config_name}</b>
⚠️ 5分钟涨跌幅连续{consecutive_count}次≤0.05%

<b>要变盘了！</b>

📈 最近{len(records)}次记录："""
    
    for record in records[-5:]:  # 只显示最近5条
        time_str = record['time']
        change = record['change_percent'] * 100
        price = record['price']
        
        # 判断K线颜色：涨为红色，跌为绿色
        if change > 0:
            color_emoji = "🟥"  # 红色方块 - 上涨
            color_text = "红色"
        elif change < 0:
            color_emoji = "🟩"  # 绿色方块 - 下跌
            color_text = "绿色"
        else:
            color_emoji = "⬜"  # 白色方块 - 平盘
            color_text = "平盘"
        
        message += f"\n• {time_str}: {change:+.3f}% {color_emoji} ({color_text}) - ${price:,.2f}"
    
    # ... 后续代码 ...
```
