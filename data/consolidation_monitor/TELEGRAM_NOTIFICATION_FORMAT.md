## 🎨 K线颜色标识

### 颜色规则
通知中的每条记录都会显示该5分钟K线的颜色：

| 图标 | 颜色 | 条件 | 说明 |
|------|------|------|------|
| 🟥 | **红色** | 收盘价 > 开盘价 | K线上涨 |
| 🟩 | **绿色** | 收盘价 < 开盘价 | K线下跌 |
| ⬜ | **白色** | 收盘价 = 开盘价 | 平盘（极少见）|

### 判断逻辑

```python
# 涨跌幅计算公式
change_percent = (close_price - open_price) / open_price

# K线颜色判断
if change_percent > 0:
    color = "🟥 红色"  # 上涨
elif change_percent < 0:
    color = "🟩 绿色"  # 下跌
else:
    color = "⬜ 平盘"  # 持平
```

- 说明触发条件

### 4. 核心提示
```
要变盘了！
```
- 简洁明确的提示
- **加粗显示**（HTML格式）

```

• 20:00: +0.032% 🟥 (红色) - $71,171.30
```

每条记录包含：
- **时间** - 5分钟K线的时间点（HH:MM）
- **涨跌幅** - 带正负号的百分比（精确到0.001%）
- **K线颜色** - 图标 + 文字说明
- **价格** - 美元计价，千分位格式化

## 🔍 详细说明

### 涨跌幅格式
```
+0.032%   # 正数 - 上涨（红色K线）
-0.045%   # 负数 - 下跌（绿色K线）
+0.000%   # 零值 - 平盘（白色K线）
```

### 价格格式
```
$71,171.30   # 千分位分隔 + 两位小数
$2,165.00    # ETH 价格示例
```

### 时间格式
```
20:00   # 北京时间 HH:MM（24小时制）
20:05   # 5分钟间隔
20:10
```

## 📡 技术实现

### Telegram Bot API
```python
url = "https://api.telegram.org/bot{TOKEN}/sendMessage"
payload = {
    'chat_id': CHAT_ID,
    'text': message,
    'parse_mode': 'HTML'  # 支持HTML格式化
```

### HTML 格式标签
- `<b>文字</b>` - 粗体
- 支持 Emoji 图标
- 不支持颜色设置（使用Emoji替代）

## 🔔 触发条件

### 基本条件
1. **横盘判断** - 5分钟涨跌幅绝对值 ≤ 0.05%
2. **连续次数** - 连续出现 ≥ 3 次
3. **首次触发** - 避免重复发送（状态管理）

### 告警时机
```
时间轴：
├─ 14:20: 横盘(1)  ← 第1次
├─ 14:25: 横盘(2)  ← 第2次
├─ 14:30: 横盘(3)  ← 🔔 触发告警！
├─ 14:35: 横盘(4)  ← 不再发送（已告警）
└─ 14:40: 波动(0)  ← 重置计数
```

```python
open_price = 71171.3

change_percent = (71194.1 - 71171.3) / 71171.3 = 0.00032 = 0.032%

# K线颜色
if 71194.1 > 71171.3:  # 收盘 > 开盘
```

## 💡 使用建议

### 1. 快速判断趋势
- **全红** 🟥🟥🟥 → 小幅上涨横盘
- **全绿** 🟩🟩🟩 → 小幅下跌横盘
- **红绿交替** 🟥🟩🟥🟩 → 震荡横盘

### 2. 结合其他指标
- 横盘 + 成交量萎缩 → 等待突破
- 横盘 + 成交量放大 → 可能酝酿大行情
- 横盘后首根大阳/大阴 → 方向确认

### 3. 交易策略
```
横盘结束后的突破方向：
- 向上突破（大阳线）→ 考虑做多
- 向下突破（大阴线）→ 考虑做空
- 继续横盘 → 继续观望
```
