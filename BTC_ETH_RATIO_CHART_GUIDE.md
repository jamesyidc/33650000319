# BTC vs ETH 强弱对比曲线图 - 使用说明

## 📊 功能简介

**BTC vs ETH强弱对比曲线图**是一个实时追踪BTC和ETH相对强弱的可视化工具，通过分析全天BTC涨跌幅大于ETH涨跌幅的比例变化，帮助您判断两者的相对走势。

## 🎯 核心功能

### 1. 比例曲线图
- **实时追踪**：显示BTC领先ETH的比例随时间的变化
- **关键阈值**：
  - 🔴 60%线：BTC明显强势
  - 🟡 50%线：平衡线
  - 🟢 40%线：ETH相对强势

### 2. 涨跌幅对比图
- 同时展示BTC和ETH的涨跌幅曲线
- 直观对比两者的价格变化
- 橙色线：BTC涨跌幅
- 蓝色线：ETH涨跌幅

### 3. 统计卡片
- **当前比例**：最新的BTC领先比例
- **最高比例**：全天BTC领先的峰值
- **平均比例**：全天的平均领先比例
- **BTC领先次数**：BTC涨跌幅超过ETH的次数

### 数据解读

#### 比例指标
- **> 60%**：🔴 BTC明显强于ETH，市场偏向BTC
- **40% - 60%**：🟡 BTC与ETH表现相近，相对平衡
- **< 40%**：🟢 ETH相对强势，市场偏向ETH

#### 实战应用
1. **趋势判断**
   - 比例持续上升 → BTC走势强于ETH
   - 比例持续下降 → ETH走势强于BTC
   - 比例波动较大 → 两者走势分化

2. **操作参考**
   - 比例突破60% → 考虑关注BTC
   - 比例跌破40% → 考虑关注ETH
   - 比例在50%附近 → 两者相对平衡

3. **配合其他指标**
   - 结合正数占比指标
   - 结合SAR偏向统计
   - 结合涨跌幅排行

## 🔧 技术细节

### 数据采集
- **采集频率**：每60秒一次
- **数据来源**：OKX交易所
- **计算方式**：
  ```
  BTC涨跌幅 = (当前价 - 开盘价) / 开盘价 × 100%
  ETH涨跌幅 = (当前价 - 开盘价) / 开盘价 × 100%
  
  如果 BTC涨跌幅 > ETH涨跌幅：
      BTC_greater_count += 1
  
  比例 = BTC_greater_count / total_count × 100%
  ```

### 数据存储
- **格式**：JSONL（每行一条JSON记录）
- **文件命名**：`btc_eth_ratio_YYYYMMDD.jsonl`
- **存储路径**：`/home/user/webapp/data/btc_eth_change_ratio/`
- **记录示例**：
  ```json
  {
    "timestamp": "2026-03-21 13:41:38",
    "beijing_date": "20260321",
    "btc_price": 70747.2,
    "eth_price": 2154.87,
    "btc_open": 69923.7,
    "eth_open": 2137.82,
    "btc_change": 1.18,
    "eth_change": 0.8,
    "btc_greater": true,
    "today_stats": {
      "total_count": 768,
      "btc_greater_count": 723,
      "ratio": 94.14
    }
  }
  ```
