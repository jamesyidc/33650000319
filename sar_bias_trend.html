🎯 系统概述

核心功能：SAR偏向趋势系统监控29种主流加密货币的SAR（抛物线转向指标）偏多/偏空状态。

  • 📊 实时统计：每5分钟统计偏多/偏空币种数量
  • 📈 趋势图表：显示历史变化趋势（双曲线）
  • 🎯 精准筛选：仅统计"2小时周期占比>80%"的币种
  • 🔄 自动更新：每5分钟自动采集，页面60秒刷新

🎉 V2.0 更新说明 - 后端计算迁移 (2026-02-16)

❌ 问题根源

用户反馈：依赖前端计算，关闭浏览器后数据丢失。

  • 表象：前端JavaScript、采集器、后端API三处重复>80%筛选逻辑
  • 问题：违反单一职责原则，数据一致性无法保证
  • 影响：维护成本高，逻辑不一致导致统计结果差异

✅ 解决方案

1. 后端API改进：

  • API /api/sar-slope/bias-ratios 直接返回筛选后的 bullish_symbols 和 bearish_symbols
  • 添加 bullish_count、bearish_count、threshold: 80 字段
  • 结果按 ratio 降序排序
  • 添加防缓存响应头

2. 前端简化：

  • 移除所有 if (ratio > 80%) 筛选逻辑
  • 直接显示后端返回的 bullish_symbols 和 bearish_symbols
  • 添加 cache: 'no-store' 和时间戳参数

3. 采集器优化：

  • 移除重复的筛选逻辑
  • 直接使用后端API返回的结果
  • 数据中添加 _computed_by: 'backend' 标记

💡 核心概念："2小时周期占比>80%"

统计方法：

  1. 每5分钟采集一次SAR数据
  2. 统计最近24条记录（24条 × 5分钟 = 2小时）
  3. 计算 position=bullish（多头）的数量
  4. 计算占比：bullish_ratio = count / 24 × 100%
  5. 如果 bullish_ratio > 80%，标记为"偏多"

物理含义：在最近2小时内，至少有1小时36分钟（80%）处于同一趋势方向。

为什么选择80%：强趋势确认，避免频繁多空切换导致的误判。

⚙️ 系统架构（5层）

┌─────────────────────────────────────────────────────┐
│         SAR偏向趋势系统 - 数据流程图                 │
└─────────────────────────────────────────────────────┘

【第1层：基础数据采集】
  PM2进程: sar-jsonl-collector
  ├─ 每5分钟采集29种币的SAR数据
  ├─ 数据源: OKX交易所15分钟K线
  ├─ 计算SAR指标值和差值
  └─ 存储: data/sar_jsonl/{SYMBOL}.jsonl
          ↓
【第2层：偏向统计采集】
  PM2进程: sar-bias-stats-collector
  ├─ 读取29种币最近24条SAR数据
  ├─ 计算bullish/bearish占比
  ├─ 筛选占比>80%的币种
  └─ 存储: data/sar_bias_stats/bias_stats_YYYYMMDD.jsonl
          ↓
【第3层：API服务】
  Flask API (端口5000)
  ├─ /api/sar-slope/bias-ratios (批量API，顶部统计)
  ├─ /api/sar-slope/bias-stats (趋势图历史数据)
  └─ /api/sar-slope/current-cycle/{symbol} (单币详情)
          ↓
【第4层：前端展示】
  页面: /sar-bias-trend
  ├─ 顶部统计卡片：当前偏多/偏空数量
  ├─ ECharts趋势图：双曲线（绿色/红色）
  ├─ 币种列表：显示具体币种和占比
  └─ 分页控制：按日期查看历史数据

💾 数据存储格式

1. SAR基础数据（JSONL格式）

文件路径: data/sar_jsonl/{SYMBOL}.jsonl

示例（BTC.jsonl）：

{"beijing_time":"2026-02-12 08:40:00","price":67345.2,"sar":66100.78,"position":"bullish","sar_diff_abs":1244.42,"sar_diff_pct":1.85}
字段名 类型 说明 示例值
beijing_time string 北京时间 2026-02-12 08:40:00
price float 当前价格 67345.2
sar float SAR指标值 66100.78
position string 周期类型 bullish / bearish
sar_diff_pct float 差值百分比 1.85

2. 偏向统计数据（JSONL格式）

文件路径: data/sar_bias_stats/bias_stats_YYYYMMDD.jsonl

示例（bias_stats_20260212.jsonl）：

{"timestamp":1707696600,"beijing_time":"2026-02-12 08:30:00","bullish_count":7,"bearish_count":0,"bullish_symbols":[{"symbol":"CFX","ratio":83.3},{"symbol":"DOGE","ratio":83.3}],"bearish_symbols":[],"total_monitored":28}

数据量统计：每5分钟1条 × 每天288条 = 约144 KB/天
